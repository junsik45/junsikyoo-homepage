This project studies the internal geometry of transformer layer updates. The main idea is to decompose a model's layer update into a predictable component and a residual component, then analyze how this residual structure changes across layers, models, prompts, and interventions.

The broader goal is to understand whether neural networks organize computation through low-dimensional update structures and whether these structures are stable under quantization, architectural change, or task shift.
